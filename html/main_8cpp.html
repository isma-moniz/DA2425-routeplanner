Detailed Description

Main driver file for the Route Planning Tool.

This file contains the main program that interacts with the user, loads location and distance data from CSV files, and calculates the best route based on different criteria like driving or walking times.

The program uses the Graph class to represent the city and calculates optimal routes based on the user's input.

Calculates the best driving route between two locations using Dijkstra's algorithm.
