Checks if a string is a numeric representation. 

This helper function determines whether a given string contains only digits.


\begin{DoxyParams}{Parameters}
{\em str} & The string to check. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
true if the string contains only digits, false otherwise.
\end{DoxyReturn}
\begin{DoxyNote}{Note}
The time complexity of this function is O(n), where n is the length of the string. The function iterates over each character in the string once to verify that they are all digits. 
\end{DoxyNote}
