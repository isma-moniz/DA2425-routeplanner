\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Edge$<$ T $>$}

\mbox{\hyperlink{classEdge}{Edge}} class representing an edge between two vertices. 

This class stores information about an edge in the graph, including the origin and destination vertices, the weights for driving and walking times, the flow on the edge, and the reverse edge for bidirectional links.
